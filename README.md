# Real-Time Client Status Tracker

A React-based dashboard that allows admins to update client application statuses and trigger real-time notifications via a Discord webhook.

## Features

- Client Dashboard
  - Displays client name, current status, and last updated time

- Admin Panel
  - Dropdown to select a new status
  - Update button to apply changes

- Notification System
  - Sends a Discord webhook when a status changes
  - Displays a temporary in-app toast notification
